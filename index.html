Technologies we work with

Play

Play is a RESTful API framework based around the familiar MVC pattern. Play is the perfect blend of power and simplicity, abstracting away most of the complexities around handling millions of requests per minute, while providing a very clean and familiar API for developers to work with.

Play has been used to build some of the most popular sites in the world, from Walmart Canada to LinkedIn. It is the perfect blend of developer productivity and power.

Akka

Akka is a distributed computing toolkit that brings resilience and complete elasticity to the JVM. Akka has been used in various domains, from video streaming, to recommendation engines, to financial trading platforms. It is a low-level framework for building the next generation of cloud-ready, cluster-based systems that require flexibility across data centers and the cloud.

Spark

Spark is a general purpose compute framework for implementing iterative processing, such as machine learning algorithms, against the largest data sets. It is currently the most popular Apache toolkit and has quickly revolutionized the big data space. It is built in Scala, while providing Scala, Java, and Python APIs for developers to work with.

Spark is a perfect fit for companies looking to modernize batch-based ETL jobs by turning them into real-time systems.

Scala

Scala is a hybrid object-oriented and functional programming language, bringing the best aspects of both approaches to build a new generation of systems built for the cloud.

While an object-oriented approach is a great way to structure applications, functional programming is the optimal way to work with data. By combining the best of both approaches on the JVM, Scala is an excellent choice for teams looking to build next generation systems.

Java

Java is one of the most popular programming languages in the world. Java 8 brings an even more refined approach to Java development with the introduction of lambda syntax, streams, and a much improved collections API.

The Lightbend platform fully supports both Java and Scala. Java is a great option when working with Play; the combination is an excellent way to introduce functional programming concepts to large Java development teams.

Kafka

Kafka is a horizontally scalable data store best applied to pub/sub and microservice architectures. It was started at LinkedIn as a way to decouple the numerous services that needed to integrate.

RedElastic heavily employs Kafka as a way to decrease complexity and increase flexibility of non-trivial enterprise architectures by reducing point-to-point integration connections. Consider Kafka your integration glue; a reasonable alternative to ESBs and other brittle integration choices.
